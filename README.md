# chastext

A repository dedicated to the world's best text editors.

Unlike other repositories on my Github account, this one is not yet a programming project. I do eventually have a plan for making my own text editor. However, at this point I have never really used vim or emacs which are supposedly the best.

I have had great success with Nano, however. It is just easy to use because all the instructions are on the screen.

Anyway, as I learn more about text editors, I will add my guides,tutorials,and reviews to this repo.

And eventually, if I gather inspiration, I might make my own text editor.
